## Phase 3: AgentLoop 工具自动注册

**新增**: `NongDiscoveredToolWrapper` — 轻量级 ITool 实现

- 包装单个 Nong 命令（如 `nong_word_read` → 实际执行 `nong word read ...`）
- 自动合并用户参数（named args → CLI flags，short keys → -o 形式）
- 委托给 NongTool.ExecuteArgsAsync 执行

**集成**: CLI 和 Web Program.cs 都加了自动发现

- NongTool 注册后异步发现全部命令并注册为独立 tool
- 发现完成后 stderr 输出 `[nong] Discovered N command tools`
- 发现失败时静默回退（不影响已有的 run_nong 工具）
