# 2026-06-13 NanoBot 应用层 Phase 7 完工

## Phase 7: Plugin 安装体系

### 新增类

PluginManager (Nanobot.Core/Skills/PluginManager.cs):
- InstallAsync(pluginName, version) — 下载 GitHub release zip, 解压到 workspace/skills/
- ListInstalled() — 读取 plugin-registry.json 返回已安装列表
- Uninstall(pluginName) — 删除目录 + 清理注册表
- CheckLatestVersionAsync(pluginName) — 查询 GitHub releases API
- 自动备份: 覆盖安装前备份旧目录

PluginInstallTool (ITool):
- 工具名: plugin_install
- 参数: plugin (required), version (optional, 不传自动查最新)
- 功能: 下载 + 解压 + 发现子 skill + 更新注册表

PluginListTool (ITool):
- 工具名: plugin_list
- 功能: 列出已安装的 plugin 及其 skill 列表

### 集成

CLI 和 Web Program.cs 启动时注册 plugin_install + plugin_list 两个工具。
